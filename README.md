# News Word Cloud

World news, word-clouded and colored by Plutchik/NRC EmoLex emotion —
auto-refreshed every 15 minutes from [The Guardian Open Platform](https://open-platform.theguardian.com/),
with no manual trigger or browser scraping involved.

Sibling project to [yt-recs-wordcloud](https://github.com/jessestrait/yt-recs-wordcloud),
reusing the same emotion-scoring approach and visual language, but built on a
self-updating GitHub Actions pipeline instead of a manually-run browser
scrape. See [CLAUDE.md](CLAUDE.md) for how the data pipeline works.

Live at https://jessestrait.github.io/news-wordcloud/
